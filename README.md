######5、如何区分类中的重载、重写、隐藏？<br>
相同点：就是大家的名称相同。<br>
重载是在同一个类中，而重写和隐藏是在基类和派生类中，这个很好区分，重点是区分重写和隐藏。<br>
（1）参数相同，如果有virtual，则为重写；如果无virtual，则为隐藏。<br>
（2）参数不同，不管有无virtual，都为隐藏。<br><br>
